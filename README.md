# Cyber Encryption Tool

Web-based encryption tool built using Flask.

## Features

- Caesar Cipher Encryption
- Caesar Cipher Decryption
- AES Encryption
- AES Decryption
- Modern Dark UI

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
py app.py
```

Open:

http://127.0.0.1:5000
